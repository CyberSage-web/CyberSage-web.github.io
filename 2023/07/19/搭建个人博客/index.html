hexo 搭建个人博客

2023年07月19日 1.2k 字 大概 5 分钟

hexo博客保姆级教程

本次我们教程分为三个部分：

  • 第一部分：软件的安装及配置
  • 第二部分：博客内容的完善与美化
  • 第三部分：gitee中部署博客

第一部分 安装git与node.js

1.安装git

Git是目前世界上最先进的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。也就是用来管理你的hexo博客文章，上传到GitHub的工具。Git非常强大，我觉得建议每个人都去了解一下。廖雪峰老师的Git教程写的非常好，大家可以了解一下git教程

2.安装node.js

Hexo是基于nodeJS编写的，所以需要安装一下nodeJs和里面的npm工具
windows：nodejs选择长期支持版

3.安装hexo

在D盘新建一个hexo文件夹并且使用git打开一次输入以下命令 

npm install hexo-cli -g

hexo init blog

cd blog

npm install

hexo s

在运行完hexo s后你会得到一个只能基于本地访问的初版个人博客网址

在完成上述步骤后你的博客已经完成50%了，接下来要做的就是更加完善我们的博客；

第二部分 博客主题安装与基本配置

你可以在hexo官网底部中找到大佬们做好的主题，我们以Ayer主题为例:

1.安装主题

在blog文件夹下右键打开git窗口，依次输入以下命令等待主题安装

git clone https://gitee.com/mirrors/ayer.git themes/ayer
hexo g
hexo s

在hexo s生成的网页中就可以看到我们安装主题后的博客了

2.修改配置主题

安装成功后在根目录下的config文件中滑到底部找到theme，修改为ayer

#Extensions
##Plugins: https://hexo.io/plugins/
##Themes: https://hexo.io/themes/
theme: ayer

然后你可以根据主题说明来修改主题配置

第三部分 gitee部署

第三部分是略微繁琐的一部分，也是考虑我们耐心的一步

我们需要注册gitee帐号并且完成实名认证(右上角-个人设置-实名认证 认证结果在俩日以内完成) 注册时填写的个人空间地址需要我们记住

1.创建个人仓库

右上方加号–新建仓库–填写信息（路径填写我们刚刚所记住的个人空间地址）–点击创建

2.上传博客文件到gitee

  • 打开刚刚创建好的仓库，复制git全局设置——在桌面本地右键打开git窗口粘贴回车

  • 然后继续在git窗口中输入下面这行命令

    SSH-keygen -t rsa -C “邮箱地址”

  • 持续回车输入下面命令获取密钥

    cat ~/.ssh/id_rsa.pub

  • 然后复制我们的密钥–打开仓库–右上角菜单中的设置–在左侧点击ssh公钥–将密钥复制进去点击确定

  • 密钥添加完成后，返回刚刚打开的git窗口输入下行命令

    ssh -T git@gitee.com

  • 回车后出现successfully代表配置成功

3.config设置

  • 在blog文件下打开config文件，滑至最底部 type后面我们输入git回车–在下一行我们输入reponsitory:空格（此处必须要有英文输入法下的空格，否则会报错）–然后来到我们刚才创建的仓库主页，点击ssh，复制后面的地址粘贴到reponsitory后面（reponsitory:英文空格ssh地址）

4.上传

在个人博客根目录下右键打开git窗口，运行下面命令 

npm install hexo-deployer-git --save

运行成功后才能部署到gitee中，然后依次输入 hexo g & hexo d

5.网址获取

打开刚才创建的仓库就可以看到我们刚才上传的代码文件

  • 点击管理将仓库开源–点击服务中的Gitee Pages服务–开启强制使用https–网站地址后面就是你的个人博客地址了
  • 当我们修改了配置时，需要在Gitee Pages中点击更新，重新获取网址

常见问题

  • 运行hexo s 出现报错提示

    这是因为你在刚才就已经打开了40000端口，我们需要点击 任务管理器–详细信息–搜索node.exe–右键点击结束任务

  • 更新后网页未发生变化
    需要我们在blog文件下右键打开git窗口依次输入回车

    hexo clean & hexo g & hexo d

这样就可以看到更新后的样式了

最后说明

搭建博客并不是一件易事，但需要我们不断尝试，终会成功！！
